## ðŸŽ¯ How to Use

### Step 1: Open Chat

Click the purple button in the bottom-left corner.

A popup will appear showing your contacts.

### Step 2: Find Someone

**Option A: Scroll through contacts**
- People with recent messages appear first
- Each shows their last message preview

**Option B: Use search**
- Type in the search box at the top
- Searches names, usernames, and emails

### Step 3: Start Chatting

1. Click on a contact
2. Type your message in the box at the bottom
3. Press **Enter** to send (or click the send button)

**Tip:** Press **Shift+Enter** for a new line without sending.

### Step 4: Navigate

- **Back arrow (â†):** Return to contact list
- **X button:** Close chat widget
- **Click outside:** Also closes the widget

## ðŸ› Troubleshooting

### Button Not Appearing?

1. **Refresh the page** (Ctrl+R or Cmd+R)
2. **Clear cache** (Ctrl+Shift+Delete)
3. **Make sure you're logged in**
4. **Try a different page**

### Can't Send Messages?

1. **Check your internet connection**
2. **Make sure message isn't empty**
3. **Refresh the page**
4. **Contact IT support**

### Popup Not Opening?

1. **Click directly on the button** (not near it)
2. **Wait a moment if page just loaded**
3. **Check browser console** (F12) for errors
4. **Report to IT if persists**

### Messages Not Updating?

1. **Popup should auto-update** every 3 seconds
2. **Close and reopen** the chat
3. **Refresh the page**
4. **Check if other users have the same issue**
